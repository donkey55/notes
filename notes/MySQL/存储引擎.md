# 存储引擎

## Mysql层次结构

* 连接层

* 服务层
  * sql接口，缓存的查询，sql分析和优化

* 引擎层
  * 真正的负责mysql的存储和提取

* 存储层
  * 存储文件系统

MySQL默认的存储引擎为InnoDB

## InnoDB

### 特点

* DML操作遵循ACID模型，支持**事务**
* **行级锁**
* 支持**外键约束**，保证数据的完整性和正确性

### 文件

xxx.ibd： xxx是表名，innoDB引擎的每张表都会对应这样一个表空间文件，存储该表结构、数据和索引。

### MyISAM

MySQL早期的默认存储引擎

不支持事务，不支持行锁，不支持外键，支持表锁

### Memory
